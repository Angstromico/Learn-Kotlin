A repository for learning and practicing Kotlin programming concepts.

## 🚀 Getting Started

### Prerequisites

- [JDK 17+](https://adoptium.net/) (or any modern JDK)
- [Kotlin](https://kotlinlang.org/) compiler

### Running the Code

Compile and run:

```bash
kotlinc src/main/kotlin/Main.kt -include-runtime -d main.jar
java -jar main.jar
```

Or run directly:

```bash
kotlin src/main/kotlin/Main.kt
```

## 📚 Learning Topics

This repository covers fundamental Kotlin concepts:

- **Basic Syntax** - Variables, functions, control flow
- **Null Safety** - Nullable types, safe calls, Elvis operator
- **Collections** - Lists, maps, sets, functional operations
- **Classes & Objects** - Data classes, objects, companion objects
- **Functional Programming** - Lambdas, higher-order functions
- **Coroutines** - Asynchronous programming basics
